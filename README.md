# Разработка интернет-приложений (РИП) 2026

## Расчет концентрации ионов в растворе после смешивания

Разработанное приложение позволит аналитикам создавать и формировать заявки на расчёты концентрации ионов для выбранных ими растворов, а экспертам завершать или отклонять данные заявки.
